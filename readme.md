# Cotizador CLC Soluciones

Aplicación web sencilla para generar cotizaciones profesionales en formato PDF, desarrollada para **C.L.C Soluciones**.

## 🚀 Características
- Inicio de sesión con usuarios:
  - Administrador: `admin` / `admin123`
  - Usuario: `ccarreno` / `123456`
- Datos preconfigurados de la empresa
- Cálculo automático de Subtotal, IVA (19%) y Total
- Agregar o eliminar ítems
- Generación y descarga de cotizaciones en PDF
- Diseño responsivo (funciona en celular y computadora)

## 📦 Cómo usar
1. Descarga o clona este repositorio
2. Abre el archivo `index.html` en cualquier navegador web
3. Inicia sesión con tus credenciales
4. Completa los datos del cliente y los productos/servicios
5. Descarga la cotización en PDF
